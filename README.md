# Java Swing Calculator

## Project Description

The Java Swing Calculator is a desktop application developed using Java Swing and AWT. It provides a simple and user-friendly interface with a modern dark theme for performing basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator also supports decimal calculations and keyboard input, allowing users to perform calculations using either the mouse or keyboard.

This project demonstrates the use of Java GUI development, event handling, object-oriented programming (OOP), and Swing components to create an interactive desktop application.

## Features

- 🌙 Modern dark theme interface
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Decimal number support
- ⌨️ Keyboard support
- 🧹 Clear (C) button
- 📱 Simple and responsive GUI

## Technologies Used

- Java
- Java Swing
- AWT
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Java-Swing-Calculator.git
   ```

2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, or VS Code).

3. Compile the program:
   ```bash
   javac Calculator.java
   ```

4. Run the application:
   ```bash
   java Calculator
   ```
